# Task Management App — Initial Requirements

## Overview

We need a task management app for our internal teams. The app should let users create tasks, assign them to people, and track progress. It needs to be web-based and easy to use.

## Goals

- Help teams stay organized
- Reduce missed deadlines
- Improve visibility into what people are working on

## Features

- Create, edit, and delete tasks
- Assign tasks to team members
- Set due dates
- Mark tasks as complete
- Comment on tasks
- Filter and search tasks
- Dashboard showing task status

## Users

Our employees will use this. Managers need to see what their team is doing.

## Technical Notes

- Should work in the browser
- We use Google Workspace so it should integrate with that somehow
- Needs to be secure
- Should be fast
